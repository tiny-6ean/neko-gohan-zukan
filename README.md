# ねこごはん図鑑｜静灯工房

猫と暮らす人のための  
**静かでシンプルなフード記録アプリ**です。

原材料・成分・完食率・食いつきの傾向を  
落ち着いて整理できるように設計しています。

---

## 🍽 アプリの特徴

### 🐾 フードの記録
名前・原材料・成分・完食率・食いつきを  
1ページで静かに記録できます。

### 🏷 タグ自動判定
原材料と成分から  
「肉系」「魚系」「豆類」「高タンパク」など  
傾向タグを自動で付与します。

### 📊 タグ別傾向グラフ
タグごとに  
完食率と食いつきの平均を可視化します。  
猫の好みの“ゆるい傾向”を静かに確認できます。

### 💾 ローカル保存
データはすべて端末内に保存されます。  
個人情報や外部送信はありません。

---

## 🌿 コンセプト

静灯工房の世界観である  
**淡色・静けさ・余白**を大切にした  
ミニマルな記録ツールです。

- 1ページで完結する軽量設計  
- スマホでも読みやすい UI  
- 猫の好みを静かに観察できる構成  
- タグとグラフで“傾向”をやさしく可視化  
- JSON保存でバックアップも可能
